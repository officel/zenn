# 前提

- 前提として pinning は、pin n ing です
- 英語の文法を勉強したことがあれば、現在分詞（動詞+ing）とか、短母音+ingは母音（n）を重ねるとかやりましたね
- それはそれとして pinning って慣用って言われるほど使われてましたかね？

# ちょっと手元で調べた

- 辞書は読むものと思っているので、うちには広辞苑のほか、小学館新選国語辞典（新撰）・三省堂国語辞典（通称三国）・新明解国語辞典（新明解）・新しい国語表記ハンドブック、使える用字用語辞典、日本語文型辞典、共同通信社新聞用字用語集記者ハンドブックなどがあるが、いずれにもピン（pin）自体が掲載されているものとないものがあった
- 当然ピニングやピンニングの用例は見つからなかった
- 個人的にはつまり慣用のレベルにないということだと認識した

つまり前提としては

> Ⅲ　撥（はつ）音,促音,長音その他に関するもの
> 1　撥音は,「ン」を用いて書く。

であり、

> 撥音を入れない慣用のある場合は,それによる。

ということになる（誤解を恐れずに言えばどっちでもいい、普段使ってるほう、多く使われているほうを正としろってことだ）

# 慣用とは

> 長い間、習慣として世間で広く使い慣らされている言葉や表現のこと

なので、

- 辞書に乗っているなら間違いなく慣用
- 普段自分が使っているならだいたい慣用（間違いもあるだろうけど）
- 普段使っておらず辞書にも掲載がない、用例が公共的な場で示されていないもの、は慣用ではない

のように考えている。

- 両方の表記があるものは揺れてるんだろうなって

# つまり（まとめ）

- pinning は事例がガイドライン等には見つけられない
- 辞書にも掲載がない、まだあまり一般的でない pinning は慣用があるとは言えない
- 外来語の表記ガイドラインに則れば、`ピンニング` と表記するのが筋ではないか
- というところに落ち着いたってワケ

# おまけ

- 別に `ピニング` って読んだり書いたりするのが嫌なわけではない
- 英語的には `/ˈpɪnɪŋ/` だから発音的にはピニンだよｗって言われてもなぁ？グはどこやったｗ
